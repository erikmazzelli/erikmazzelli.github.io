# Erik Mazzelli — Portfolio

Portfolio pessoal desenvolvido para apresentar projetos, experiências e experimentos front-end.

🔗 **Live:** https://erikmazzelli.github.io

---

## Sobre

Este projeto funciona como vitrine profissional e laboratório técnico.  
É mantido com foco em:

- Código limpo e legível
- Estrutura escalável
- Performance
- Responsividade (mobile-first)
- Manutenção simples e evolutiva

---

## Stack

- HTML5
- CSS3 (Flexbox / Grid)
- JavaScript (ES6+)
- GitHub Pages (deploy)

---
